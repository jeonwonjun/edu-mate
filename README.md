# EduMate
**“공부는 했지만, 진짜 이해했는지는 모르겠는 순간을 위한 동료.”**

EduMate는 학습자 스스로 복습하고 회고할 수 있도록 돕는 교육용 피드백 챗봇입니다.

## 프로젝트 개요
EduMate는 학습 과정의 “이해”와 “성장”에 초점을 맞춘 Java 기반 콘솔 챗봇 프로젝트입니다.
사용자가 공부한 내용을 간단히 입력하면, EduMate는 이를 기반으로 질문을 던지고 피드백을 제공하여 복습을 유도합니다.

이 프로젝트는 우아한테크코스 프리코스 4~5주차 자유 미션으로 진행되며,
지난 3주 동안 배운 객체지향 설계, Stream API, 예외 처리 등을 실제 서비스 형태로 적용하는 것을 목표로 합니다.

## 페르소나
| 이름 | 나이 | 상황 | 필요 |
| --- | --- | --- | --- |
| 지은 | 23세 | 프로그래밍을 처음 배우는 대학생 | 공부는 했지만 개념을 정확히 이해했는지 스스로 점검하기 어려움 |
| 민수 | 27세 | 이직 준비 중인 주니어 개발자 | 학습한 내용을 복습하고, 누락된 부분을 피드백받으며 체계적으로 공부하고 싶음 |

EduMate는 이 두 사람처럼 **“공부는 했지만, 내 것이 되었는지 확신이 없는 사람”** 을 위한 학습 동반자입니다.

## 목표 (복습·피드백 챗봇)

1. **복습 지원:**
- 사용자가 학습 내용을 입력하면 EduMate가 핵심 개념을 기반으로 질문을 던져 스스로 내용을 설명하고 이해를 점검할 수 있도록 돕습니다.
    
2. **피드백 제공:**
- 사용자의 답변을 바탕으로 기본적인 피드백(정확성, 빠진 개념 등)을 제공합니다. (추후 AI API를 활용한 자연스러운 피드백으로 확장 예정)
    
3. **학습 회고 유도:**
- 하루 학습을 마친 후, EduMate가 회고 질문을 통해 “무엇을 배웠고, 무엇이 어려웠는가?”를 돌아보게 합니다.
